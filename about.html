About this website:

This website presents a randomly selected philosophical quote of the day using a free public API. You can moreover discover thoughts by thinkers across time, filtered by philosopher or keyword.

The project was built using only HTML, CSS, and JavaScript. It demonstrates the use of asynchronous API calls, DOM manipulation, and responsive design.
