About

This website presents a randomly selected philosophical quote of the day using a free public API. You can discover thoughts by great thinkers across time, filtered by philosopher, historical period, or theme.

The project was built using only HTML, CSS, and JavaScript, and demonstrates the use of asynchronous API calls, DOM manipulation, and responsive design.
